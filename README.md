# SQL Sales & Customer Analytics Using MySQL

## 📌 Project Overview

This project showcases a collection of practical SQL queries designed to analyze sales, customer behavior, product performance, and business operations using a relational database in MySQL. The database consists of multiple related tables, including **Customers**, **Orders**, **OrderItems**, and **Products**, allowing complex business questions to be answered through SQL joins, aggregations, filtering, grouping, sorting, and analytical functions. The project demonstrates how SQL can transform raw transactional data into meaningful business insights by identifying top customers, measuring sales performance, evaluating product demand, and analyzing customer purchasing patterns. It is an ideal portfolio project for aspiring Data Analysts looking to demonstrate proficiency in SQL-based business intelligence and relational database querying.

The project uses a relational database containing the following tables:

* **Customers**
* **Orders**

These tables are connected through primary and foreign keys, enabling efficient querying across multiple business entities.

## 📈 Business Questions Solved

The analysis answers several real-world business questions, including:

* Calculate the total amount spent by each customer.
* Identify customers who have placed multiple orders.
* Find the top three customers based on total purchase value.
* Calculate the total quantity of products sold by category.
* Compute the total value of every order using product price and quantity.
* Identify the highest-spending customer.
* Calculate the average order value for each customer.
* Find customers who joined before 2022 and have made purchases.
* Identify products sold more than once.
* Retrieve the latest purchase date for each customer.

## 💡 Key Insights

This project demonstrates how SQL can be used to analyze transactional business data and support decision-making. By combining customer, product, and order information through relational joins, the analysis identifies high-value customers, measures revenue contribution, evaluates product demand across categories, and tracks customer purchasing behavior over time. The queries illustrate common analytical tasks performed by data analysts in e-commerce, retail, and sales environments, making this project highly relevant for real-world business analytics.
